Family Feud — How To Host a Game

Be sure to review ALL parts of this page before hosting a game.

The Game Board

  1. This is the board you show on the screen for both teams to see
    • This should NOT be on the same screen as the Admin View
  2. Before the start of the game, be sure to enter "Family Names"
    • Click on the name to enter edit mode. Then type in the name
  3. Once you start a game, it will give you a Game Code
    • This is the same code you must use for the Admin View and Host View
    • See details about these Views in their respective sections below.
  4. After the "Face Off", be sure to select which team will PLAY
    • The game won't let you continue revealing answers until you do.
    • NOTE:If nobody gives an answer, and the host has asked each possible player, use the option for "No Correct Answers Given", and neither team will get the points.
  5. Users should have 10 seconds to give an answer once the question is asked to them
    • Use the "control" key on the keyboard to start (and stop) the timer
    • If it gets to zero (0) and they have not given an answer, you give the team a STRIKE
  6. When users give an answer, you will compare to the survey answers using the Admin View.
    • More details on this in the Admin View section below.
  7. If you determine a user's answer is correct, press the number on your keyboard that corresponds to the survey answer on the board
    • EXAMPLE: if their answer corresponds with #3 on the board, you press the #3 key on your keyboard.
  8. If you determine the user's answer is incorect, you can press the "escape" key on your keyboard, to give them a STRIKE
  9. Once a team has won the main rounds, you can click on the Fast Money button at the bottom of this page.

  10. Use these keyboard shortcuts when on the game board:

The Admin View

  1. This View shows you the current question(s) as well as the related answer(s)
    • This is how you can determine if a player's answer is correct
  2. Do not open this View on the same screen that is showing the Game Board page.
    • You would be revealing all the answers.
    • You should open this up on your phone or another device.
  3. You will need to enter the same Game Code that is displayed on the Game Board.
    • This ensures you are pulling in the right answers
  4. After starting a new game, or after each round, be sure to press the REFRESH button to load the latest question and answers.
  5. If you determine a user's answer is correct, press the number on your keyboard that corresponds to the survey answer on the board
    • EXAMPLE: if their answer corresponds with #3 on the board, you press the #3 key on your keyboard.
  6. Once a team has won the main rounds, you can click on the Fast Money button to load the questions & answers for "Fast Money"
    • You will have to press SELECT QUESTIONS to load the questions.
  7. When you are ready to start a new game, just click on the New Game button.

The Host View

  1. This View shows you the current question during each main round
    • If you are using a separate host (ie. your "Steve Harvey"), they would use this View
    • They should pull this view up on their phone or another device.
  2. The Host will need to enter the same Game Code that is displayed on the Game Board.
    • This ensures they are pulling in the right questions
  3. After starting a new game, or after each round, they should press the REFRESH button to load the latest question and answers.
  4. Once a team has won the main rounds, they can click on the Fast Money button to show the questions & answers for "Fast Money"
    • NOTE:The host should wait until after the Admin "selects" the questions before loading them. Otherwise, they could be pulling in the wrong questions.
  5. When you are ready to start a new game, just click on the New Game button.

The Face Off View

  1. You have two ways to handle the "Face Off" part of the game via this site
    • Use the "Face Off" popup on the Game Board page.
    • Use a separate screen just for the "Face Off"
  2. If you use the popup option on the Game Board page:
    • Click on the word "Face Off" that is just above the answer table
    • A popup screen will appear, with the two team names automatically showing side-by-side
    • The participants in the Face Off can then hit the appropriate keys to BUZZ for their team (see keys below).
  3. If you use a separate screen just for the "Face Off", you can open the Face Off View on that separate screen.
    • Once you enter the team names, you will see them appear side-by-side
    • The participants in the Face Off can then hit the appropriate keys to BUZZ for their team (see keys below).
  4. For either option, you will need to make sure there is an extra keyboard attached to the computer showing the the "Face Off"

Fast Money

  1. On the "Fast Money" page, you will enter the answers that users give to the host during fast money
  2. Once you click "Player One", a a timer will appear, and the cells one the left side of the table will become editable.
    • If you want, you can click on the numbers in the timer to edit the value (ie. give them a bit more time)
    • Be sure to start the timer AFTER the host finishes the first question.
  3. You will then need to enter the player's answer quickly!
    • After you enter the player's answer, hit the TAB key to go to the next entry
    • You'll notice that the cell you just left wills how ANSWERED
  4. If the player does not give an answer, or they run out of time, just enter a single space as their answer (ie. press the spacebar).
  5. To reveal their answers, click on the word "ANSWERED" for the respective question
    • This will show their answer, as well as cause the cell next to it to start blinking red
    • Once the cell is blinking red, you can enter the score score they get for their answer.
    • NOTE: You won't be able to see the score you typed until you hit the ENTER key. Be sure to type in only digits (or it will throw an error)
    • If you made a mistake and need to change it, just click on the score, and confirm the change.
  6. When you are ready for "Player Two", be sure to Hide Player One answers by clicking on the button just above the table.
    • After "Player Two" has returned to the area and is no longer facing the board, you can click on Show Player One button to show the answers again.
    • This will be useful so you can easily double check if "Player Two" gives a duplicate answer.
  7. When "Player Two" is going, you will once again have to enter the answers quickly!
    • This time, you will have to pay attention for duplicate answers.
    • Be sure to use the appropriate keyboard shortcut for the duplicate answer (se below)

  8. Use these keys when on the Fast Money page.
